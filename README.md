# Appointment Booking UI

## Description
A responsive appointment booking user interface built using HTML, CSS, and Bootstrap. 
The project allows users to select services, choose appointment slots, and submit booking details (UI only).

## Features
- Appointment booking form layout
- Service selection section
- Date and time slot UI
- Responsive design using Bootstrap

## Tech Stack
HTML, CSS, Bootstrap

## How to Run
1. Download or clone the repository
2. Open index.html in a browser

## Future Enhancements
- Backend integration using Django
- Database to store appointment details
